    One-time 5-minute Azure setup. Your password is never stored — Microsoft handles login securely.

    1

    Go to Azure Portal

    Open portal.azure.com → sign in with your Microsoft/OneDrive account.

    2

    Register a New App

    Search "App registrations" → "New registration" → Name: HexamechQuotations → Supported account types: "Accounts in any organizational directory and personal Microsoft accounts" → Click Register.

    3

    Add Redirect URI

    In your app → Authentication → Add a platform → Single-page application → Redirect URI:

    http://localhost:8080/hexamech-quotation.html

    Enable "Access tokens" and "ID tokens" → Click Configure.

    4

    Add Files.ReadWrite Permission

    API permissions → Add a permission → Microsoft Graph → Delegated → expand Files → tick Files.ReadWrite → Add permissions.

    5

    Paste your Client ID below

    App Overview → copy Application (client) ID
